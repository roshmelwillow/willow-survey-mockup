# Willow Patient UX Survey Prototype

This is a Vite + React wrapper around the Claude-generated `App.jsx` prototype.

## Deploy to Vercel

1. Upload these files to GitHub with `package.json` at the repo root.
2. In Vercel: Add New → Project → Import your repo.
3. Use:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
4. Deploy and open the generated `.vercel.app` URL.
